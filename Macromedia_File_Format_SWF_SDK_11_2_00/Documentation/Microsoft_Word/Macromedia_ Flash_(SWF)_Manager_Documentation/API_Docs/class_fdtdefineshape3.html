Detailed Description

Create a DefineShape3 tag.

Creates a Flash 3.0 Define Shape tag. All the DefineShape tags only differ in the fill/line style arrays and fill/line style records (accepts alpha color values).

See also:
FFillStyle, FLineStyle, FShapeRec, FShapeWStyle, see also FExampleRectangle.cpp
