Detailed Description

Set the values of the transform - same as FCXForm with alpha channel.

Do we need the _has parameters? Is 0 a valid value for each of these mult or add params?

Parameters:
U32 - _hasAdd: Has color addition values if equal to 1
U32 - _hasMult: Has color multipy values if equal to 1
S32 - _redMultTerm: Red multiply value
S32 - _greenMultTerm: Green multiply value
S32 - _blueMultiTerm: Blue multiply value
S32 - _alphaMultTerm: Alpha transparency multiply value
S32 - _redAddTerm: Red addition value
S32 - _greenAddTerm: Green addition value
S32 - _blueAddTerm: Blue addition value
S32 - _alphaAddTerm: Alpha transparency addition value
