Macromedia Flash File Format (SWF) - Documentation Overview

 

The Macromedia Flash File Format (SWF) documentation is provided in both HTML and Microsoft Word formats. Documentation is provided for both the Macromedia Flash File Format (SWF) File, explaining how it works, and for the Macromedia Flash File Format (SWF) manager, which gives programmers an API with which to write Macromedia Flash (SWF) files.

 

The directories are laid out as follows:

Documentation

 

HTML  

Documentation in HTML format.

 

Macromedia Flash File Format (SWF) SDK Documentation

Documentation in 14 files, describing the Macromedia Flash File Format (SWF) and how it works. Each file discusses some aspect of  Macromedia Flash (SWF), such as bitmaps, morphing, etc. The file “intro.htm” contains an overview and links to the other files.

 

Macromedia Flash (SWF) Manager Documentation

The file SDK.htm gives an overview to the developer’s toolkits, and how to use them.

 

API_Docs

The full API and all its methods, in hyperlinked HTML. The API document begins at “index.html.”

 

Microsoft Word

Documentation in the HTML format.

 

Macromedia Flash File Format (SWF) SDK Documentation

Documentation, in 14 files, describing the SWF file format and how it works. Each file discusses some aspect of SWF, such as bitmaps, morphing, etc. The file “intro.doc” contains an overview and links to the other files.

 

Macromedia Flash (SWF) Manager Documentation

The file SDK.doc gives an overview to the developers toolkits, and how to use them.

 

API_Docs

The full API and all its methods, in hyperlinked HTML. The API document begins at “index.html.”

 

 

Please ignore directories that end in *_files, as these merely contain support data for the other documents.
