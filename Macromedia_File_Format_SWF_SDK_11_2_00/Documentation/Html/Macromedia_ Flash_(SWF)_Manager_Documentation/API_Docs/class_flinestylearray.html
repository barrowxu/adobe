Detailed Description

An array of line styles.

LineStyles are held in arrays of LineStyles which are held in DefineShape Tags.

See also:
FLineStyle, FExampleRectangle.cpp

U32 FLineStyleArray::Add (FALineStyle * lineStyle)

Add the line style to the array.

Automatically handles the case of packing the corresponding SWF object properly when you go over 255 styles. Not that we can see why you'd do this.

Parameters:
lineStyle - the style object to add to the internal array

U32 FLineStyleArray::Size (void)

The number of items in the list.

This is needed when a containing DefineShape needs to know how how many styles there are and therefore how many bits are needed for an index. The given line style is added to the end of the line style array. The position of the added line style is returned so that the line style can later be referenced.

void FLineStyleArray::WriteToSWFStream (FSWFStream * _SWFStream)

Write the array to the output stream.

Writes to the stream by travelling through all of the nodes in the array and writing their line styles. First has to write the count of line style arrays. Sees if count is small enough to fit in to an 8 bit field, and either writes the count into an 8 bit field, or writes all 1's into an 8 bit field and writes the real count into a 16 bit field.
