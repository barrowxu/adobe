Detailed Description

Holds an array of ShapeRec's.

The ShapeRecs are written to a SWFStream in the order they are added to the array. Normally used to store Font Glyphs in a Definefont tag. An array of ShapeRec'sThis should probably be called FShapeArray or such. FShapeWStyle is an expanded version.
