HFButton::HFButton (HFShape * shape1, HFShape * shape2, HFShape * shape3, HFShape * shape4)

Creates a button using four HFShape objects, one for each button-state.

This overloaded button constructor allows sophisticated-looking buttons to be created, with one complete HFShape object for each of the four button-states specified explicitly.

Parameters:
shape1 - Appearance of up-state.
shape2 - Appearance of over-state.
shape3 - Appearance of down-state.
shape4 - Appearance of hit-state, which is always invisible.
