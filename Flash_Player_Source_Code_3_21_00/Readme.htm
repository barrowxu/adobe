Macromedia Flash Player Source Code SDK

Version 1.0

March 16, 2000

Overview

Welcome to the Macromedia Flash Player Source Code SDK or 'SDK'. The SDK contains a modified version of the Macromedia Flash Player designed to help a third party easily port the Macromedia Flash Player to any platform or product, as well as the documentation to perform this task. The code in the SDK is a partial rewrite of the Macromedia Flash Player version 4 code, modified for ease of porting and platform independence.

 

The code as provided compiles on a Win32 system.

Compiling

A Visual C++ project file is provided. The code is ready to compile with one caveat: if you are not on a system that supports Unicode Cut and Paste, comment out:

 

#define EDITTEXT_CLIPBOARD_WIDE

 

in global.h. (In the very likely case you don’t know, compile the project and if you get an error that CF_UNICODETEXT is undefined, then comment out EDITTEXT_CLIPBOARD_WIDE.)

 
After you compile, the Flash player should run from your C++ IDE.

Documentation

Documentation is provided in the Documentation subdirectory. We recommend starting with Design.htm (or design.doc in the Docs in MSWord directory.)
