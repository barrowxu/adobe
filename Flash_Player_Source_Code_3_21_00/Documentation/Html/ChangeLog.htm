Changes from Version 0.9 to 1.0

·        Added $version variable in version.h

·        Version code in splayer.cpp

Changes from Beta 3 to Version 0.9

·        Enhanced menu support.

·        Pop-up (right mouse button) menu support.

·        Documentation.

·        Minor cleanup and bugs.

Changes from Beta 2 to Beta 3

·        Sound. Implemented and documented.

·        Cut and Paste support, for both Unicode and ASCII

·        Switched from bool back to BOOL. There were too many problems with native support of bool – a better solution on many fronts.

·        Assorted bugs.

·        Some code cleanup.

Changes from Beta 1 to Beta 2

·        Streaming. An interface for files streaming is implemented.  The interface mimics the Netscape calls in order to provide a common reference point for both plug-ins and stand alone players. The streaming interface is a part of the NativePlayerWnd class.

·        File.h and File.cpp removed. Deprectated as a result of the new streaming interface.

·        Edit Text is now restored and implemented as core code.

·        Bugs in the control menu and MenuStatus structure are fixed.

·        Window redraw code is fixed. The window is no longer completely redrawn with every frame.

·        Fixed size windows tested and documented.
