Macromedia Ported Flash Test Suite

 

What's included?

HTML Acceptance Level Test Suite- f4playertesthtml.hqx or f4playertesthtml.zip-This test suite is designed to ensure that all the basic features of Flash are functioning correctly with your Flash Player. This test suite was designed with web browser testing in mind and the suite is navigated through HTML. If you are not testing with HTML capabilities then you will need to use the SWF only Acceptance Level Test Suite.

SWF only Acceptance Level Test Suite -f4playertestswf.hqx or f4playertestswf.zip -This test suite is designed to work with a Ported Flash Player without HTML capability. The test suite is designed to ensure that all basic features of Flash Player are functioning correctly with your Ported Flash Player.

Inside each hqx or zip file you will find:

  1. an index.html file describing all the tests and giving their location or a link to the start of the test.
  2. a checklist.html listing each test in a table. This table has a pass or fail field and a comment field. Hopefully you will just need to check the pass field. If for some reason your ported player does not pass please check the fail field an provide an explanation in the comment field.
  3. the test files. Use the index.html to find out which files to use. Original fla files were not included to help with download size. If you need these files for some reason use the contact information below.
